# Idea to Product: Architectural perspective Talk

Materials and resources for "Idea to Product: Architectural perspective" talk, held in NTP, Novi Sad, 27. 6. 2024.

## Running slides

```console
npm install -g reveal-md
reveal-md talk.md
```
